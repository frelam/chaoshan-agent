# 潮汕美食 — 用户贡献指南

> 你的每一份真实评价，都在让这份美食地图变得更公平、更丰富。
> 不论好评差评，只要是真实的，都是宝贵的。

---

## 一、先了解你的身份

在写评价前，请提供一些基本信息。这些信息不会公开你的具体身份，但能帮助我们做出更个性化的推荐。

### 必填信息

| 字段 | 说明 | 示例 |
|------|------|------|
| `type` | 你的身份类型 | `local`（本地人）/ `tourist`（游客）/ `returning-visitor`（回访客） |
| `hometown` | 你来自哪里（精确到城市） | `汕头` / `广州` / `北京` |
| `age_range` | 年龄段 | `18-25` / `26-35` / `36-50` / `50+` |

### 可选信息（强烈建议提供）

| 字段 | 说明 | 示例 |
|------|------|------|
| `taste_tags` | 你对自己的口味描述 | `["喜欢油香", "喜欢弹牙"]` |
| `visit_context` | 用餐场景 | "周六午餐，带外地朋友" |

> **为什么需要这些信息？** 因为"好吃"的定义因人而异。一位汕头本地人和一位北京游客对同一家店的评价可能完全不同——这份差异不是错误，恰恰是信息丰富的表现。有了你的身份画像，我们才能告诉其他人"和你相似的人觉得这家店怎么样"。

## 二、怎样写一条好评价

### 最低标准

- **不少于 30 字**
- **包含具体口感/味道描述**
- **必须亲自去过、吃过的店**

### 一篇好评价的要素

```
✅ 好的评价：
  "脖仁油脂分布均匀，入口即化。38元一盘份量不错。
   吊龙也嫩，但今天的切工一般，有几片稍厚。
   总体来说是汕头牛肉火锅的标杆，会再来。"

❌ 差的评价：
  "好吃，推荐。"
  "太绝了家人们，yyds！"
  "还行吧。"
```

### 评价要素清单

| 要素 | 重要程度 | 说明 |
|------|---------|------|
| **店名 + 地址** | ⭐️必填 | 让其他人能找到 |
| **点了什么菜** | ⭐️必填 | 最好每道菜分别评价 |
| **口感描述** | ⭐️必填 | 嫩/弹/脆/韧/滑……具体描述 |
| **价格信息** | ⭐️推荐 | "一份XX多少钱" |
| **和谁一起去** | 👍加分 | 一个人吃 / 朋友聚餐 / 带家人 |
| **和别家对比** | 👍加分 | "比XX家的更嫩" / "比上次吃的XX差远了" |
| **是否愿意再去** | ⭐️推荐 | 以及为什么 |

## 三、评分指南（1-5分）

| 分数 | 含义 | 什么时候给这个分 |
|------|------|------------------|
| **5分** | 惊艳 | 超出预期，强烈推荐。会特地再来。 |
| **4分** | 满意 | 符合预期，值得吃。会考虑再来。 |
| **3分** | 一般 | 中规中矩，无过无失。不会特地来，但也不排斥。 |
| **2分** | 失望 | 低于预期，有明显缺点。不太可能再来了。 |
| **1分** | 难吃 | 严重问题（食材不新鲜/做法不对/态度恶劣）。建议别人也避雷。 |

> **关于3分：** 3分不是差评！3分意味着"还行，但不算特别"。一个普通的店也应该得到真实的3分，而不是"给个面子打4分"。

### 菜品评分 vs 综合评分

- **菜品评分**：对单道菜的评价（更细粒度，更有参考价值）
- **综合评分**：本次用餐的整体体验（包含环境、服务等非食物因素）

理想情况下，两道菜评分分别写，同时给一个综合评分。

## 四、常见场景的评分参考

### 牛肉火锅

```
5分 → 脖仁入口即化，有明显雪花纹理，38元物超所值
4分 → 肉新鲜，切工好，味道稳定
3分 → 普通牛肉火锅水平，不难吃也不惊艳
2分 → 肉不新鲜/切太厚/汤底没味道
1分 → 肉有异味/明显不是当天宰杀
```

### 蚝烙/蚝煎

```
5分 → 蚝仔饱满大颗，番薯粉比例完美，外脆里嫩
4分 → 蚝仔新鲜，煎得金黄酥脆
3分 → 中规中矩，蚝仔偏小
2分 → 粉太多/蚝仔太少/煎得太油
1分 → 蚝仔不新鲜/用冷冻蚝
```

### 肠粉

```
5分 → 皮薄馅靓，酱汁刚好，米香浓郁
4分 → 皮滑馅足，味道不错
3分 → 普通肠粉水平
2分 → 皮太厚/馅太少/酱汁太咸或太淡
1分 → 皮过硬/馅不新鲜
```

## 五、争议处理

### 当你和别人的感受不一样

这是完全正常的！不同人的口味差异正是我们最珍视的价值。

**正确的做法：**
- 说清楚为什么不一样（"我觉得胸口朥太油了，可能我本来就不太吃肥肉"）
- 描述你的感受，而不是否定别人的感受
- 提供对比参照（"之前吃过XX家的，觉得那家更好吃"）

**不需要做的：**
- 不用"纠正"别人的评分
- 不用担心你的评价"拉低"了某家店的分数
- 不需要为了"合群"打高分

### 关于"踩雷"评价

如果你吃到的和网上的好评完全不一样，**请一定写下来**。这种信息差恰恰是最有价值的——它说明：
1. 店铺水平不稳定（有时好吃有时不好吃）
2. 不同人的口味差异巨大
3. 这家店的营销可能大于实际

## 八、你贡献的价值

每一条真实评价都在做什么：

```
你的评价
  ├─ 帮助其他人找到适合他们的店
  ├─ 让算法的个性化推荐更准确
  ├─ 平衡营销推广的声音（真实的差评比好评更珍贵）
  └─ 记录下这些店铺在这个时代的真实面貌
```

谢谢你的贡献 ❤️

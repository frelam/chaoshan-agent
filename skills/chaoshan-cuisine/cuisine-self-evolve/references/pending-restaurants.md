每一条新店草稿按以下格式记录：

```markdown

**来源**: web_search / 小红书 / 知乎 / 其他
**搜索查询**: "触发该发现的具体搜索词"

**基本信息**:
- 店名: 
- 地址: 
- 区县: 
- 品类: 
- 来源描述: "搜索到的原始描述摘要"

**评价信息（提取自搜索，注明来源）**:
- reviewer_type: local / tourist / 未知
- 推荐菜: 
- 正面评价: 
- 负面评价: 
- 价格参考: 
- 初步可信度: 高 / 中 / 低（见 search-sources.md 判定标准）

**交叉验证**:
- [ ] 是否有多个来源提到？
- [ ] 是否有本地人推荐？
- [ ] 是否有近期（<3个月）评价？
- [ ] 是否有任何营销推广信号？

**备注**:
```

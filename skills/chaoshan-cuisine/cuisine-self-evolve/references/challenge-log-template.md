## 归档位置

所有挑战日志按日期存储在:

```
data/challenge-logs/
├── challenge-001-2026-06-11.md
├── challenge-002-2026-06-18.md
└── ...
```

## 日志完整性检查清单

在每个挑战日志生成后，检查以下内容：

- [ ] 是否包含所有 6 个 Agent 的参与记录？
- [ ] 每个议题的论据是否完整引用？
- [ ] 辩论摘要是否至少记录了 2 轮交互？
- [ ] 共识投票结果是否记录？
- [ ] Valid Bug 是否都附带修复方案？
- [ ] 算法变更是否已实际应用到 FAIRNESS-ALGORITHM.md？
- [ ] 版本号是否已 bump？

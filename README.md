# 🏮 Chaoshan Agent — 潮汕文化 Agent（双引擎自演进）

> 一个面向潮汕文化传承的 AI Agent 技能集合，以 **Hermes Agent Skill** 为载体，提供潮汕话翻译、潮汕美食导航等服务。  
> **这是一个会自己长大的项目** — 白天有人用，凌晨自己学，每周还有多Agent辩论优化算法。

## ♻️ 双引擎自演进

本项目有两个完全独立的自演进流水线，跑在 Hermes Agent 的 cron 上：

### 引擎一：潮汕话翻译（每日词条学习）

每天 5 次（07/10/13/16/20 时），每次自动从网络搜索 5 个潮汕话翻译对，自测验证后追加知识库：

```
搜索发现 → 查重过滤 → 自测验证 → 学习更新 → 同步源码 → GitHub提交
  
每周一 04:00 → 再加一轮 consolidation（去碎片化总结提炼）
```

> 🎯 日增量：~25 条新词，月增量：~500 条候选

### 引擎二：潮汕美食（每周双轮驱动）

#### 📅 周一 06:00 — 数据收集轮

```
web_search × 4-6 种查询
  → 提取新评价/新店铺/状态变更
  → 营销推广识别 → 可信评价追加到评价库
  → 评价者画像同步更新
  → git push
```

#### 📅 周四 06:00 — 公平算法挑战赛

```
6个不同立场的Agent挑战公平算法：
  🧓 本地老饕    🧳 外地游客   📊 数据科学家
  🎭 营销号操盘手  📖 文化研究者  ⚖️ 公平性审计员

挑战提案 → 3轮辩论 → 共识投票 → 算法优化 → 更新文档 → git push
```

> 🎯 每周自动发现数据盲点并优化算法公平性

## 🍜 潮汕美食（v3.0）核心特色

### 不做榜单，不做排名

同一家店对汕头本地人和北京游客的意义完全不同。所以不展示固定排名——根据每个人的画像（家乡/年龄/口味偏好）做个性化推荐。

### 7 层公平保障

| 层级 | 机制 | 作用 |
|------|------|------|
| 1️⃣ | 零商业收入 | 不接受任何商家付费或推广 |
| 2️⃣ | 开源透明 | 全部数据和算法在 GitHub 上可查可验证 |
| 3️⃣ | 中位数评分 | 抗极端评分，刷分无效 |
| 4️⃣ | 少数声音保留 | ≥15% 的观点独立列出 |
| 5️⃣ | 人群分层 | 本地人/游客/各年龄段分开展示 |
| 6️⃣ | 反操控检测 | 自动识别刷分集群并降权 |
| 7️⃣ | 权重上限 | 不存在"大 V 一条顶十条" |

### 多Agent算法挑战赛

每周四，6 个不同立场的 Agent（本地老饕/外地游客/数据科学家/营销号操盘手/文化研究者/公平性审计员）各自阅读算法文档和数据，提出挑战、辩论、达成共识，持续优化算法公平性。

## 🔄 潮汕话翻译（v1.5+）特色

- 🔄 **双向翻译** — 潮汕话 ↔ 普通话，保留语气、语序、文化内涵
- 🔊 **潮州拼音（Peng'im）** — 所有潮汕话输出带 1-8 调号标注
- 📖 **语法解析** — 语序转换、11种否定词辨析、程度副词转换
- 🏮 **文化注释** — 俗语典故、工夫茶文化、文白异读、区域变体
- 📚 **知识库** — 120+ 词汇、38+ 俗语、完整语法体系

```bash

cd chaoshan-agent
bash install.sh
```

安装后重启 Hermes Agent 即可使用。

## 🚀 使用方式

### 潮汕话翻译

```
用户：瓦爱来去踢桃
Agent：我想去玩
       拼音：ua2 ain3 lai5 ke3 ti1 tho5
       「踢桃」是有音无字词，借音写法……
```

### 潮汕美食导航

```
> 汕头哪家牛肉火锅好吃？
> 推荐吃的 — 我是北京来的，第一次来汕头
> 脖仁是什么部位？和吊龙有什么区别？
> 胸口朥在日本叫什么？
```

### 在 Claude Code 中使用

```bash
claude -p "$(cat teochew-prompt.txt)" -f "需要翻译的文本"
```
